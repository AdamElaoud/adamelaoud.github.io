Apollo Dec. 2020
Kotlin JSON
Apollo is a gamification of "Go-No-Go" inhibition testing in an Android app.
The user is prompted quickly with two types of prompts - "Go" prompts, which instructs the user to tap the screen, and "No-Go" prompts, which instructs the user to avoid tapping the screen. The test collects data on how well users appropriately perform both actions in a rapid fire stream of 10 prompts.
